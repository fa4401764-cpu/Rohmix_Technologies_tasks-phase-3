# Project 1: Data Wrangling & Feature Engineering

## Overview

This project focuses on combining multiple datasets and creating new calculated features to prepare data for analysis and predictive modeling. The project demonstrates data cleaning, merging, transformation, and feature engineering techniques using Python and Pandas.

## Objectives

* Merge multiple datasets such as sales, products, and customer information.
* Clean and preprocess data.
* Handle missing values and duplicates.
* Create new business metrics and calculated features.
* Prepare a structured dataset for further analysis.

## Technologies Used

* Python
* Pandas
* NumPy
* Jupyter Notebook

## Features Created

* Total Revenue
* Profit Margin
* Customer Lifetime Value (CLV)
* Additional derived business metrics
   
## Workflow
1. Load datasets.                                                                            
2. Inspect data quality.                                                        
3. Clean missing and inconsistent values.                                                       
4. Merge datasets using common keys.                                                                
5. Create new calculated features.                                                                                             
6. Perform exploratory analysis.                                                                                      
7. Save the processed dataset.                                                                                       
## Results

The final dataset contains cleaned, integrated, and enriched information that can be used for predictive analytics and business intelligence applications.

# Project 2: A/B Testing & Hypothesis Testing

## Overview

This project demonstrates how statistical methods can be used to evaluate business decisions through A/B testing. A synthetic dataset was generated to simulate the performance of two marketing campaigns: Campaign A and Campaign B.

## Objectives

* Generate and analyze campaign performance data.
* Compare conversion rates between two campaigns.
* Apply hypothesis testing techniques.
* Identify statistically significant differences in campaign performance.

## Technologies Used

* Python
* Pandas
* NumPy
* SciPy
* Statsmodels
* Matplotlib
* Seaborn
* Jupyter Notebook

## Statistical Methods Used

### Independent T-Test

Used to compare the mean conversion rates of Campaign A and Campaign B.

### Chi-Square Test

Used to determine whether campaign type and conversion outcomes are associated.

### ANOVA Test

Used to analyze variance between campaign groups and verify performance differences.

## Workflow

1. Generate synthetic campaign data.
2. Perform exploratory data analysis.
3. Calculate conversion rates.
4. Conduct Independent T-Test.
5. Conduct Chi-Square Test.
6. Conduct ANOVA Test.
7. Interpret statistical results.
8. Draw business conclusions.

## Results

The analysis identifies whether there is a statistically significant difference between the two campaigns and determines which campaign performs better.
